# 🚀 Low-Level Systems & Silicon Design Journey


Welcome to my core systems engineering repository! This space serves as public, verifiable proof of my engineering growth, deep-diving into the intersection of low-level firmware and hardware design. 

As an Electronics & VLSI student, I am utilizing this 60-day summer sprint to move past high-level abstractions and master how software directly manipulates silicon registers.

## 🗺️ Journey Roadmap

### 📁 Phase 1: Bitwise Mastery & Memory Architecture (Days 1–20)
* High-performance bit manipulation in pure C (`&`, `|`, `^`, `~`, `<<`, `>>`).
* Solving hardware-logic tracking puzzles on LeetCode and HackerRank.
* **Goal**: Complete fluency in pointer arithmetic and register masking.

### 📁 Phase 2: Bare-Metal Embedded Systems (Days 21–40)
* Programming ARM Cortex-M architecture via register manipulation.
Simulating hardware protocols (`I2C`, `SPI`, `UART`) using **Wokwi**.
* Direct data manipulation using device datasheets without high-level library abstractions.

### 📁 Phase 3: Silicon Logic & Verilog (Days 41–60)
* Translating digital logic electronics into hardware design.
* Solving core hardware description puzzles on **HDLBits**.
* Simulating foundational digital components on **EDA Playground**.
